# Document Flow Bot - Система документооборота

## Описание проекта

Интеллектуальная система для автоматической обработки и валидации документов на основе продукционной модели (Rule-Based Logic).

## Тема проекта

**Документооборот (Офисный бот)**

Система решает следующие задачи:
- Валидация бланков (проверка дат, заполненности полей)
- Извлечение сущностей (NER): имя, дата, ИНН
- Анализ структуры документов
- Поиск дубликатов
- Автоматическая сортировка и экспорт в Excel

## Функциональность (v1.0 - Rule-Based)

### Реализованные правила валидации:

1. **Критические проверки (Hard Filters)**
   - Наличие обязательных полей
   - Проверка формата даты
   - Валидация ИНН

2. **Бизнес-логика**
   - Проверка срока действия документа
   - Валидация диапазонов числовых значений
   - Проверка на запрещенные значения (blacklist)

3. **Предупреждения**
   - Приближающийся срок истечения
   - Отсутствие необязательных полей
